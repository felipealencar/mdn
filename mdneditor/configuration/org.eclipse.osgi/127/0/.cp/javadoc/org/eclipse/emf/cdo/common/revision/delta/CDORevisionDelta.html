org.eclipse.emf.cdo.common.revision.delta

Interface CDORevisionDelta

  • All Superinterfaces:
    CDOBranchVersion, CDOIDAndVersion, CDOIdentifiable, CDORevisionKey
    All Known Subinterfaces:
    InternalCDORevisionDelta

    public interface CDORevisionDelta
extends CDORevisionKey
    Represents the change delta between two revisions of a CDO object. The detailed feature deltas are returned by the getFeatureDeltas() method.
    Since:
    2.0
    No Implement
    This interface is not intended to be implemented by clients.
    No Extend
    This interface is not intended to be extended by clients.
