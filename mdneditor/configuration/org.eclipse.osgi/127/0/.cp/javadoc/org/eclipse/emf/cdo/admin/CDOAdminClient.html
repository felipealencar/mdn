Interface CDOAdminClient

  • All Superinterfaces:

    public interface CDOAdminClient
extends CDOAdmin
    A client-side administrative interface.

    It can fire the following events:

    • CDOAdminClient.ConnectionStateChangedEvent after the connection state has changed.
    No Implement
    This interface is not intended to be implemented by clients.
    No Extend
    This interface is not intended to be extended by clients.
