Package org.eclipse.emf.cdo.common.lob

Common concepts for dealing with large binary objects and large character objects.

See: Description

  • Interface Summary 
    Interface Description
    CDOLobHandler
    A callback interface for handling large objects.
    CDOLobStore
    Stores and loads large objects.
  • Class Summary 
    Class Description
    CDOBlob
    A identifiable binary large object with streaming support.
    CDOBlobOutputStream
    An OutputStream that produces a CDOBlob.
    CDOClob
    A identifiable character large object with streaming support.
    CDOClobWriter
    A Writer that produces a CDOClob.
    CDOLob<IO>
    A identifiable large object with streaming support.
    CDOLobInfo
    Encapsulates ID and size of a large object.
    CDOLobStore.Delegating
    An abstract large object store that delegates all method calls to a delegate.
    CDOLobUtil
    Some useful helpers for dealing with large objects.
